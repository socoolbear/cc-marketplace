# cc-marketplace

개인용 Claude Code 플러그인 마켓플레이스.

## 설치

```bash
/plugin marketplace add socoolbear/cc-marketplace
```

## 플러그인 목록

| 플러그인 | 스킬 | 설명 |
|---------|------|------|
| harness-engineering | `/harness-engineering:setup` | 프로젝트에 하네스 엔지니어링 인프라 구축 (지식 아키텍처, 기계적 강제, 3-에이전트 파이프라인, 품질 추적, 실패 방지) |
| documents | `/documents:resume` | 경력기술서, 이력서를 MD 파일로 작성 |

## 플러그인 설치

```bash
/plugin install harness-engineering@cc-marketplace
/plugin install documents@cc-marketplace
```
